## Get Started

```
 # 下载依赖
 npm i vite-vue3-cli -g

 # 更新版本
 npm update vite-vue3-cli -g

 # mac 确保使用管理员权限下载
 sudo npm i -g vite-vue3-cli --unsafe-perm=true 
 
 # 创建项目
 create-cli 
```
